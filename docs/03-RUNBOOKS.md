## RB-01: 헬스체크 실패

### 증상

- `/health` 비정상 또는 타임아웃.

### 즉시 조치

1. `REL-2026-05-02.3` 배포 직후인지 확인 → 롤백 후보.
2. `curl -sf http://127.0.0.1:8080/health` (포트는 `.env`).
3. `docker compose restart` (데모).

### 채움 예시 (INC-2026-0412 일부)

| 시각 | 내용 |
|------|------|
| 14:02 | 알림: Blackbox `DL-PORTAL` 헬스 2회 연속 실패 |
