# 데이터 분류 · 보존 · 마스킹 — 데모랩

> 가상 정책 표입니다. 실제 법무·DPO 검토를 대체하지 않습니다.

## 분류 등급

| 등급 | 정의 | 예시(가상) |
|------|------|-------------|
| 공개 | 외부 공개 가능 | 마케팅 페이지, 공개 API 문서 |
| 내부 | 사내·파트너 계약 범위 | 주문 ID, 배송 상태 |
| 민감 | 식별·재식별 가능 | 이메일, 전화, 결제 식별자 |
| 제한 | 법적·계약상 엄격 | 카드 PAN(보관 금지 원칙), 건강정보 등 |

## 보존 기간 (Lab 예시)

| 데이터 유형 | 등급 | 보존 | 삭제/익명화 |
|-------------|------|------|-------------|
| 액세스 로그 | 내부 | 90일 | 이후 집계 통계만 |
| 주문 본문 | 내부 | 3년(가상) | 세금 규정 가정 |
| 감사 로그 | 내부 | 1년 | 분쟁 시 연장 규칙 별도 |
| 디버그 덤프 | 민감 | **7일 이내** | 자동 삭제 |

## 마스킹·취급

| 필드 | 로그·대시보드 표시 |
