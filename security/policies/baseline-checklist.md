# 서버 보안 기준선 체크리스트 (요약)

운영 환경에 붙여 쓰기 전, 조직 표준에 맞게 수정하세요.

## OS · 접근

- [ ] 최신 보안 패치 적용 주기 합의
- [ ] SSH: 키 기반, root 직접 로그인 비활성화(정책 허용 시)
- [ ] 방화벽: 필요 포트만 개방, 관리망 분리

## 서비스

- [ ] 비root 사용자로 프로세스 실행(컨테이너 포함)
- [ ] TLS 종료 및 최소 암호 스위트(리버스 프록시 또는 서비스 메시)

## 로그 · 감사

- [ ] 시간 동기화(NTP/Chrony)
- [ ] 관리자 명령 감사 로그 보존 기간

## 백업 · 복구

- [ ] RPO/RTO 문서화
- [ ] 분기별 복구 리허설
